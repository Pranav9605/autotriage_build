# AutoTriage Setup

Welcome to the unified `autotriage_build` repository. This repository combines both the frontend and backend components of the AutoTriage platform.

## Architecture

* **`frontend/`**: Contains the React/Vite based user interface, built with TypeScript. Used for viewing triage results, ITSM integrations, and incident management.
* **`backend/`**: Contains the core logic for the LLM-powered triage engine, synthetic ticket generator, RAG pipeline, and backend services.

## Prerequisites

Ensure you have the following installed on your machine:
* Node.js (v18+)
* Python (v3.10+)
* PostgreSQL (with pgvector extension, if applicable)
* Git

## Setting up the Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Setting up the Backend
